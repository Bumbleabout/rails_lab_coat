Rails Lab Coat
==============

Tools for doing rails science! (and cleaning up after yourself)

Requires:
* VirtualBox
* Vagrant

kickstart:
* ./build.sh
* move ./rails_lab_coat anywhere you want
* in rails_lab_coat
** vagrant up
** vagrant ssh
** cd /vagrant
** rails new my_lab
** rails server
** http://localhost:3000
